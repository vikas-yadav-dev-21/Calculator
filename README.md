# Calculator APP

A beautiful, dark-themed calculator built with Flutter. Unlike simple calculators, this app supports complex mathematical expressions, order of operations (BODMAS), and nested parentheses.

## ‚ú® Features

* **Dark Mode UI:** A sleek, eye-friendly dark interface similar to iOS/Android native styles.
* **Complex Expressions:** Supports parentheses `()` and operator precedence (e.g., `7x(8+(9/3))`).
* **Smart Editing:**
    * `DEL`: Remove the last character.
    * `AC`: All Clear to reset the screen.
* **Real-time Logic:**Accurate floating-point arithmetic.

## üöÄ Getting Started

Follow these steps to run the calculator locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/vikas-yadav-dev-21/calculator-app.git](https://github.com/vikas-yadav-dev-21/calculator-app.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd calculator-app
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
